# ShopHub

A modern e-commerce scaffold.

## Tech Stack
- Next.js 16
- Supabase
- MeiliSearch
- Zustand
- Framer Motion

## Setup Instructions

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Environment Variables:**
   Copy `.env.example` to `.env.local` and fill in your keys.

3. **Development:**
   ```bash
   npm run dev
   ```

*Note: Backend integrations are modular and replaceable. Ensure you do not commit any credentials or secrets.*
